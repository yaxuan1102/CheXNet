# CheXNet
## Binary Classification of Pneumonia


### Data
A total of 5863 chest radiography images in JPEG format.

|              | Normal | Pnemonia |
|--------------|--------|----------|
|     Train    |  1341  |   3875   |
|     Test     |  234   |    390   |
|  Validation  |  8     |    8     |

### Model
- Source: https://github.com/zoogzog/chexnet
- Framework: PyTorch
